# KeaSports

KeaSports é um sistema web desenvolvido em **Django** para gerenciamento de corridas, participantes, resultados e arquivos relacionados. Possui integração com **Tailwind CSS** para estilização e utiliza **PostgreSQL** como banco de dados principal.

---

## Tecnologias Utilizadas

- **Backend:** Python 3 + Django
- **Banco de Dados:** PostgreSQL
- **Frontend:** Tailwind CSS
- **Gerenciamento de Pacotes:** npm (para Tailwind)
- **Controle de Versão:** Git / GitHub
