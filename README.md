# KeaSports

KeaSports é um sistema web desenvolvido em **Django** para gerenciamento de corridas, participantes, resultados e arquivos relacionados. Possui integração com **Tailwind CSS** para estilização e utiliza **PostgreSQL** como banco de dados principal.

---

## Tecnologias Utilizadas

- **Backend:** Python 3 + Django
- **Banco de Dados:** PostgreSQL
- **Frontend:** Tailwind CSS
- **Gerenciamento de Pacotes:** npm (para Tailwind)
- **Controle de Versão:** Git / GitHub

---

## Funcionalidades

- **Gerenciamento de Corridas:** criar, editar e listar corridas com datas, categorias e distâncias.  
- **Cadastro de Participantes:** registrar participantes com dados pessoais e associá-los a corridas.  
- **Inscrição em Corridas:** participantes podem se inscrever nas corridas disponíveis.  
- **Gestão de Resultados:** registrar resultados, tempos, colocações e classificações.  
- **Upload de Arquivos:** importar planilhas Excel com dados de participantes ou resultados.  
- **Filtragem e Paginação:** listagens de corridas e participantes filtráveis e paginadas.  
- **Templates Personalizados:** visualização organizada com HTML customizado.
